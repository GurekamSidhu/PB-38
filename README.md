# PB38

Price Setting API

### Server setup

Setup and run server (on windows)

Install flask

```
pip install flask
```

Navigate to /PB-38

Linux and Mac:

```
export FLASK_APP=flaskr
export FLASK_ENV=development
flask run
```

Windows cmd:

```
set FLASK_APP=flaskr
set FLASK_ENV=development
flask run
```

Windows PowerShell:

```
$env:FLASK_APP = "flaskr"
$env:FLASK_ENV = "development"
flask run
```
